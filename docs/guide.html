快速开始

本地开发

go run ./cmd/buildwebs233-server -config server.yaml
cd ui
npm install
npm run dev

默认登录信息

后台地址：`http://127.0.0.1:6640/admin`

账号密码：`root / root`

自动化流水线

`CI` 工作流会自动执行 `go vet`、`go test ./...`、`npm run typecheck` 和 `npm run build`。

`Docs Pages` 会把 `docs/` 发布到 GitHub Pages。

`Release` 会在推送 `v*` tag 时生成 Windows 和 Linux 预编译包。
