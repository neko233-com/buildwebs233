Go 1.26 建站引擎与后台管理文档

面向快速建站、模板复用、低代码编辑和中国地区备案流程的服务端平台。

快速开始 产品路线 GitHub 仓库

核心能力

  • 默认 `6640` 端口后台管理
  • `server.yaml` 配置与热重载
  • 页面/模板文件化存储，可平滑迁移到数据库
  • SSE 驱动 HTML 与配置刷新

自动化

  • GitHub Actions 运行 `go vet`、`go test`、前端构建与工作流校验
  • GitHub Pages 自动发布文档站
  • Tag 发布自动打包 Linux/Windows 预构建二进制

扩展建议

  • 多租户：`tenant_id` 隔离页面、模板、站点配置
  • 模板市场：元数据入库，资源上 OSS/CDN
  • 审核流：按地区接入备案材料与审批状态
